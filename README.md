# Todo Backend

This project is aimed at allowing users to keep track of their todos

## Website URL

`https://tm-backend.herokuapp.com`

## Local Hosting

To run this app locally, take the following steps 
1. `clone the repo`
2. run `npm i`
3. run `npm run start:dev`

## Technologies

1. NodeJs (JavaScript)
2. MongoDB (Online Mongo Atlass)
